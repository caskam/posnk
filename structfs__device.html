Detailed Description

An instance of a filesystem driver.

This structure describes an instance of a filesystem driver, it is returned by a filesystem's mount function and contains information on the interface provided by the driver aswell as the mounted device. In order to implement a filesystem driver you must provide two things:

  • A mount function of the form fs_device_t fs_mount( dev_t device, uin32_t flags);
  • Implementations of the callbacks in fs_device_operations

For more information on the callbacks see the documentation of fs_device_operations
