Detailed Description

Describes a pointer to a stream.

A stream pointer is what is handled by most programs, it is referred to by a unique handle (fd) and refers to a stream. The actual state of a stream is stored in the stream info object, the stream pointer contains only flags affecting its behaviour on process events
